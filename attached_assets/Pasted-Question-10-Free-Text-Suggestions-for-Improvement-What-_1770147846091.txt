5. CSV Storage Requirements

File name: survey_responses.csv

If the file does not exist, create it with headers.

Append new responses without overwriting existing data.

CSV columns must be in this exact order:

timestamp
name
company
overall_experience
service_quality
timeliness
communication
professionalism
issue_resolution
ease_of_access
value_added_by_security_advisor
efficiency_of_security_advisor
suggestions


Timestamp should be saved in UTC format.

6. Backend API Endpoints

Create the following Flask routes:

GET /

Serves the main survey webpage.

POST /submit

Accepts JSON data from the frontend.

Validates required fields.

Saves data into the CSV file.

Returns success response.

GET /download-csv

Returns the CSV file as a downloadable attachment.

Make the URL simple and easy to access.
